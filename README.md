Project Title: A Web-Based Automotive Parts and Accessories Product Showcase for Rapidé in San Jose, Antique
Designer: Brian Rich V. Sarmogenes / BSIT-3B
Short Overview of the Project:
Rapidé is a web-based product showcase system designed for an automotive parts and accessories shop located in San Jose, Antique. The system provides customers with an easy way to browse and explore available products online, improving the shop's digital presence and customer accessibility without the need for a physical visit. The platform highlights the shop's catalog in a clean, organized, and visually appealing interface.
Framework Used: HTML, CSS, Bootstrap — developed using Visual Studio Code
